Key User Flows

Parent/Child — Free Play

drawintheair.com/ → /play → WaveToWake onboarding → Activity menu → Gameplay
No auth required. No account needed. Camera stays local.

Teacher — Sign Up & First Session

drawintheair.com/for-teachers → app.drawintheair.com/auth/login → Google OAuth → /auth/callback → /dashboard → /classroom/start → /classroom/[id]/lobby
Auto-creates teacher record with 5-day trial. No credit card required.

Teacher — Class Mode (Main Site)

drawintheair.com/class → Google OAuth (implicit) → /class (dashboard) → Pick activity → /class/lobby → /class/round → /class/results
Lightweight class mode on main site. Students join via drawintheair.com/join.

Student — Join Class Session

drawintheair.com/join → Enter 4-digit code → Enter first name → /join/play → Gameplay (real-time)
No account, no email, no login. Just code + name.

Teacher — Upgrade to Pro

/dashboard/upgrade → POST /api/stripe/checkout → Stripe Checkout → /dashboard?upgraded=true
Webhook updates tier to 'pro'. Cancel returns to /dashboard/upgrade.

School — Onboard Teachers

/school/teachers/invite → POST /api/school/invite → Teacher receives invite link → POST /api/auth/join-school → Teacher gets Pro via school
Token-based invite. Email must match. Auto-grants Pro tier via school membership.

SEO Discovery → Conversion

Google search → drawintheair.com/trace-a → /play (try activity) → app.drawintheair.com/auth/login → /dashboard
~80+ SEO landing pages feed organic traffic into the play → signup funnel.
